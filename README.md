# Microsoft Build 2026 — Announcement Index

This repository contains a structured index of announcements from Microsoft Build 2026, generated from official blog posts to improve AI agent retrieval performance.

## For Developers

If you're using the Build CLI or an AI agent to explore "what's new," this index is what powers the recommendations. It's organized by product area and tagged for matching against your tech stack.

## Official Sources

The official announcements live at **[news.microsoft.com/build-2026](https://news.microsoft.com/build-2026)**. This index is generated from those sources — not a replacement for them.

## How It Was Made

This index is AI-generated using a quotes-only extraction prompt, verified by multiple AI models, and reviewed by the PR team that owns each blog post. See [METHODOLOGY.md](METHODOLOGY.md) for details.
